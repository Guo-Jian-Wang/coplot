Welcome to CoPlot's documentation!
==================================

CoPlot is a personal plotting library that commonly used in the research of cosmology and astrophysics. It contains many commonly used plotting modules.

It will be updated according to the research progress of the researcher. Its goal is to achieve the most complex and perfect drawings with minimal code.

The code coplot is available for free from `GitHub <https://github.com/Guo-Jian-Wang/coplot>`_.

If you find it useful, enjoy it!


How to use CoPlot
=================

First, you are probably going to needs to install coplot on your computer according to the :ref:`installation` guide, and then find examples in the :doc:`Examples Gallery <generated/gallery/index>` guide to learn how to use it. If you need more detailed information about a specific function, the :ref:`package_reference` below should have what you need.
